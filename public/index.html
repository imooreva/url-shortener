URL Shortener Microservice

Pass a valid URL and the app generates a shortened URL using six alphanumeric characters.

Pass a valid shortened URL and the app will redirect your browser to the original URL.

Usage Methods & Examples:




Create shortened link in URL:
[project_url]/api/new/http://www.google.com

Retrieve shortened link via URL:
[project_url]/api/abcxyz
