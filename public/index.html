URL Shortener Microservice

Pass a valid URL and the app generates a shortened URL using six alphanumeric characters. An object containing the original URL and shortened URL should be returned.

Pass a valid shortened URL and the app will redirect your browser to the original.

Usage Examples:

Create a new shortened URL:
https://peaceful-citadel-24973.herokuapp.com/api/new/http://www.google.com

Use a shortened URL:
https://peaceful-citadel-24973.herokuapp.com/api/abcxyz
