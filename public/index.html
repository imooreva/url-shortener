URL Shortener

Pass a valid URL as a parameter and the app will create a shortened URL using five alphanumeric characters. The app should return a JSON object containing the original URL and shortened URL.

Pass a valid shortened URL and the app will redirect your browser to the original.

Usage Examples:

Create a new shortened URL:
{app url}/new/http://www.google.com

Use a shortened URL:
{app url}/aBcDe
